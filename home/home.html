Apache2
What is that?

Apache2 is a web server that can run on many operating systems (Unix,BSD,Linux,Microsoft Windows and Novell Netware and other platforms) that is useful for serving and functioning web sites. The protocol used to service the web/www uses HTTP.

Nginx
What is that?

NGINX is a web server that also functions as an email proxy, reverse proxy, and load balancer. This software structure asynchronous and possible avant drivers many request are processed at the same time.

Nginx
Apa itu ?

NGINX adalah web server yang juga berfungsi sebagai email proxy, reverse proxy, dan load balancer. Struktur software ini bersifat asinkron dan event-driven; yang memungkinkan banyak request atau permintaan diproses pada waktu bersamaan.
